# Mappe Kepler.gl

Questa repository raccoglie alcune mappe interattive realizzate con **Kepler.gl** e pubblicate online tramite **GitHub Pages**.

Il progetto ha lo scopo di rendere consultabili e condivisibili mappe tematiche in formato web, accessibili direttamente da browser senza necessità di installare software GIS.

## Link alla mappa online

La mappa è consultabile al seguente indirizzo:

https://brunocorda.github.io/mappe-kepler/

## Contenuto della repository

La repository contiene file HTML esportati da Kepler.gl e pubblicati tramite GitHub Pages.
